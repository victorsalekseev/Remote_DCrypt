Remote DCrypter v 1.0.0.0
Шифрующая программа для безопасного хранения важных файлов на удаленном сервере.

Программа шифрует содержимое файла и его имя, затем данные передаются по защищенному каналу SSH2 на удаленный компьютер (сервер), где и они хранятся.
Таким образом, доступ к файлам будет только у вас и только из программы Remote DCrypter.

Программа не требует установки, имеет маленький размер 512 кБ и может запускаться с любого носителя (дискета, компакт-диск, флешка).

Настройки программы (ключи доступа, данные авторизации на сервере) хранятся в зашифрованном файле "keys.key".

Для начала работы необходимо указать:
на вкладке "Соединение": Сервер, Каталог, Пользователь (ВНИМАНИЕ! Вводите каталог относительно корневой папки SSH-подключения);
на вкладке "Шифрование": Пароль для шифрования имен файлов, пароль для шифрования файлов и длину ключа шифрования. Используя введенные пароли, программа сгенерирует ключи шифрования;
на вкладке "Общие" можно интегрировать программу в контекстное меню проводника для удобной отправки файлов.

После сохранения настроек зайдите в меню "Действия"-->"Список файлов на сервере" (или просто отправьте любой файл из левого окна программы двойным щелчком мыши по нему или из соответсвующего контекстного меню). При первом запросе программа спросит пароль для соединения с сервером. Введите его, нажмите ОК. Через несколько секунд установится соединение и Вы сможете работать со своими файлами.

В качестве сервера можете использовать практически любой. Главное, чтобы на нем был организован доступ по SSH2. Программа корректно работает с http://rootkits.ru/hosting/. При создании сайта нужно выбрать тип веб-сервера "UNIX / Apache". Воспользуйтесь конструктором тарифного плана, чтобы подобрать экономичное решение (от 22 рублей в месяц).

Примечание: 
1. Если Вы забудете свои пароли на шифрование файлов, восстановить файлы будет невозможно. Если пароль не верен, то появляется окно с текстом "Ошибка: плохие данные", а в правом списке вместо имени файла отображается "Имя файла не расшифровано".
2. Для работы программы требуется пакет Microsoft(R) .NET Framework 2.0 и выше. Если он еще не установлен, его можно бесплатно скачать с сайта Microsoft: http://www.microsoft.com/downloads/details.aspx?displaylang=ru&FamilyID=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5
